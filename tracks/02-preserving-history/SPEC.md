# Griot

**Pitch:** Scrappy notes & photos + Gemini → a digital museum exhibit for the neighborhood.

## What you'll build (45 minutes)
Upload or type out raw, messy notes from a family journal or oral history interview. The AI will instantly process the text, extract key dates to build a historical timeline, map out coordinates of lost landmarks, and draft a compelling narrative summary for a community archive.

## The moment it clicks
*"It extracted the exact year my grandfather's shop opened from that messy journal entry."*

## Things to think about
- **Focus on the timeline today.** Don't worry about building a full interactive map yet; just get the coordinates and dates parsing correctly.
- **Your signature detail.** Will your app have a formal "academic archivist" tone, or a warmer "storytelling elder" tone? Set the system prompt accordingly.
- **Test with fictional lore.** Feed it a made-up story about a neighborhood superhero to see how well the AI structures a timeline.

## Where to take it after the jam (Polished Version)
- Interactive map integration (Google Maps API) to pin historical sites
- Audio recording feature to transcribe oral histories live
- Image generation to "restore" descriptions of old buildings
- QR code generator to print physical plaques for the neighborhood
- Database integration to combine multiple residents' stories

## Tech you'll touch
- Antigravity (AI-driven IDE)
- Python + google-genai SDK (backend)
- Google Gemini API (Entity Extraction & Timeline formatting)
- HTML / CSS / JS (frontend)
